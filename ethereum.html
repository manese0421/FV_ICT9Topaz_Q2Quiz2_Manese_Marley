what is ethereum?

Introduced by Vitalik Buterin in 2015, Ethereum was created to change the Bitcoin community by introducing programmable transactions. Ethereum is a decentralized platform that aids developers to make smart transactions with DApps.

key features of ethereum

Feature Description
smart contracts Self-executing contracts with the terms of the agreement directly written into code.
decentralized applications (DApps) Applications that run on a peer-to-peer network instead of centralized servers.
